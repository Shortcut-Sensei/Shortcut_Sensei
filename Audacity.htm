Top 85 Shortcuts for Audacity

Discover the Power of Audacity Shortcuts

Audacity is a powerful, free, open-source audio editing software that is widely used by professionals and hobbyists alike. Whether you are editing podcasts, creating music, or fine-tuning audio recordings, knowing the right shortcuts can significantly speed up your workflow and enhance your productivity.

On this page, we've compiled a comprehensive list of the top 85 shortcuts for Audacity, categorized for your convenience. Mastering these shortcuts will help you navigate through the software more efficiently and focus on what truly mattersâ€”your creativity.


Maximize Your Efficiency with Audacity Shortcuts
Our curated list of Audacity shortcuts covers everything from basic navigation to advanced editing techniques. By familiarizing yourself with these shortcuts, you can:

  • Save Time: Perform common tasks quickly without relying on menus.
  • Boost Productivity: Streamline your editing process and focus more on creative aspects.
  • Enhance Workflow:Easily switch between different tools and functions.
  • Improve Accuracy:Reduce the chances of errors by minimizing mouse usage.


Navigation Shortcuts

These shortcuts are designed to help you navigate through your audio tracks quickly and efficiently. They allow you to move the cursor, zoom in and out, and focus on specific sections of your project.

Key Combination Description
Home Move Cursor to Start
End Move Cursor to End
Left Arrow Move Cursor Left
Right Arrow Move Cursor Right
Ctrl + Left Arrow Move Cursor to Previous Label
Ctrl + Right Arrow Move Cursor to Next Label
Ctrl + 1 Zoom In
Ctrl + 2 Zoom Normal
Ctrl + 3 Zoom Out
Ctrl + F Zoom to Selection

Editing Shortcuts

Editing shortcuts streamline the editing process by providing quick actions for common tasks like cutting, copying, pasting, deleting, undoing, and redoing. They help you manipulate audio clips and arrangements with ease.

Key Combination Description
Ctrl + X Cut
Ctrl + C Copy
Ctrl + V Paste
Delete Delete
Ctrl + Z Undo
Ctrl + Y Redo
Ctrl + A Select All
Ctrl + T Select Track
Ctrl + B Select Region
Shift + End Select Cursor to End
Shift + Home Select Cursor to Start
Shift + Left Arrow Extend Selection Left
Shift + Right Arrow Extend Selection Right
Ctrl + Shift + Left Arrow Shrink Selection Left
Ctrl + Shift + Right Arrow Shrink Selection Right

Playback Shortcuts

Playback shortcuts facilitate playback control, allowing you to play, pause, stop, and adjust playback speed. They also include features like loop playback, scrubbing, and playing from selection to enhance your editing workflow.

Key Combination Description
Spacebar Play/Pause
Ctrl + Spacebar Stop
Shift + Spacebar Play from Selection
Shift + L Loop Playback
Shift + S Toggle Scrubbing
Shift + Up Arrow Forward Playback
Shift + Down Arrow Reverse Playback
Ctrl + Up Arrow Increase Playback Speed
Ctrl + Down Arrow Decrease Playback Speed

Recording Shortcuts

Recording shortcuts are essential for starting, pausing, and stopping recording sessions swiftly. They may also include features like sound-activated recording, enabling you to capture audio efficiently.

Key Combination Description
R Record
Shift + R Toggle Sound Activated Recording
P Pause Recording
S Stop Recording

Track Control Shortcuts

Track control shortcuts help manage individual audio tracks within your project. They enable actions such as muting, soloing, adjusting track height, syncing tracks, and toggling track panels for better organization and control.

Key Combination Description
M Toggle Mute
S Toggle Solo
H Toggle Track Height
T Minimize/Maximize Track Height
Shift + L Toggle Sync-Lock Tracks
Ctrl + P Toggle Track Panel

Time Shift Tool Shortcuts

These shortcuts are specific to the time shift tool, which allows you to move audio clips horizontally along the timeline. They enable precise adjustments to the timing and placement of audio elements within your project.

Key Combination Description
Ctrl + Shift + Left Arrow Move Clip Left
Ctrl + Shift + Right Arrow Move Clip Right
Ctrl + Left Arrow Nudge Clip Left
Ctrl + Right Arrow Nudge Clip Right

Envelope Tool Shortcuts

Envelope tool shortcuts are tailored to the manipulation of volume and other envelope points within your audio tracks. They facilitate tasks such as creating, deleting, adjusting, and resetting envelope points for dynamic audio editing.

Key Combination Description
Ctrl + Click Create Envelope Point
Ctrl + Shift + Click Delete Envelope Point
Alt + Up Arrow Adjust Envelope Up
Alt + Down Arrow Adjust Envelope Down
Alt + Left Arrow Adjust Envelope Left
Alt + Right Arrow Adjust Envelope Right
Ctrl + Alt + Click Reset Envelope Point

Custom Shortcut

The custom shortcut category provides an option for users to define their own personalized shortcuts based on their specific preferences and workflow requirements. It allows for flexibility and customization in utilizing Audacity's features.

Key Combination Description
Ctrl + Alt + Shift + X Custom Shortcut (Replace X with desired key)

Other Useful Shortcuts

This category includes miscellaneous shortcuts for accessing preferences, exporting and importing audio, saving and opening projects, and managing the Audacity interface. These shortcuts enhance overall workflow efficiency and user experience.

Key Combination Description
F12 Toggle Toolbar
Ctrl + P Preferences

Ctrl + Shift + I Import Audio
Ctrl + S Save Project
Ctrl + Shift + O Open Recent Project
Ctrl + N New Project
Ctrl + W Close Current Project
